Часто задаваемые вопросы о расписании:

Вопрос: Когда проходят лекции по математическому анализу?
Ответ: Лекции по математическому анализу проходят в понедельник (10:00-11:20 для потока С1.1, 14:30-15:50 для потока В1.1, 16:00-17:20 для потока Ю1.1), во вторник (14:30-15:50 для потока Ю1.2, 16:00-17:20 для потока В1.2) и в среду (13:00-14:20 для потока З1.2, 14:30-15:50 для потока С1.2).

Вопрос: Кто ведет лекции по математическому анализу?
Ответ: Лекции по математическому анализу ведут преподаватели: Скубачевский (поток С1.1 и С1.2), Городецкий (поток З1.1, В1.1, З1.2 и В1.2) и Глухов (поток Ю1.1 и Ю1.2).

Вопрос: Где проходят семинары по дискретной математике в четверг?
Ответ: В четверг семинары по дискретной математике проходят: 11:30-12:50 (Дениско в В506, Зайцева в В504), 13:00-14:20 (Дениско в В506, Мухин в В504) и 14:30-15:50 (Соколова в В512, Зайцева в В502, Дениско в В506, Мухин в В504).

Вопрос: Какие занятия проходят в среду после обеда?
Ответ: В среду после обеда проходят: 13:00-14:20 (лекция MiA, лекция по матану, занятия по введению в экономику), 14:30-15:50 (лекция по матану, занятия по введению в экономику, семинары), 16:00-17:20 (семинары, занятия по ОПР и основам финансов) и 17:30-18:50 (лекция по основам финансов, ОПР Go, семинары).

- 10:00-11:20: Основы промышленной разработки (Python и Java), Основы финансов (лекция), Коммуникация
- 11:30-12:50: Английский язык, Искусственный интеллект (лекция)
- 13:00-14:20: Искусственный интеллект (лекция), Английский язык
- 14:30-15:50: Английский язык, лекции по матану и MiA
- 16:00-17:20: Английский язык, семинары по матану, ОПР (Python и Java), лекция по матану
- 17:30-18:50: Семинары по матану

В: Как найти занятие по имени преподавателя?
Ответ: Укажите имя преподавателя в своем вопросе. Например: "Когда занятия у преподавателя Иванова?"

В: Как узнать расписание на конкретный день?
Ответ: Укажите день недели в своем вопросе. Например: "Какие занятия в понедельник?"

В: Как найти расписание для конкретной аудитории?
Ответ: Укажите номер аудитории в своем вопросе. Например: "Какие занятия проходят в аудитории 101?"

В: Когда проходят занятия по определённому предмету?
Ответ: Укажите название предмета в своем вопросе. Например: "Когда проходят занятия по математике?"

В: Что означают направления "Север", "Юг", "Запад", "Восток"?
Ответ: Это обозначения разных аудиторий или корпусов, где одновременно могут проходить занятия в одно и то же время.

В: Как получить полное расписание на неделю?
Ответ: Спросите "Покажи полное расписание на неделю" или "Полное расписание".

В: Могут ли быть изменения в расписании?
Ответ: Да, расписание может меняться. Эта версия расписания актуальна на текущий семестр. Для получения информации об изменениях обращайтесь в учебный отдел.
